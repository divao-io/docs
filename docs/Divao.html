    verify(jwt) → {JsonWebTokenPayload}

    Verifies that the JSON Web Token:

    • is issued for the correct audience, i.e. that aud matches document.location.hostname
    • is not expired, i.e. that exp is after the current system time
    • is not used too early, i.e. that nbf is before the current system time

    System time comparisons tolerate a clock skew of up to 2 minutes.

    Parameters

    • jwt String

      the JSON Web Token string.

    Returns

    • JsonWebTokenPayload

      the JSON Web Token payload.

    Example

    const payload = window.Divao.verify(jwt);
console.log(payload.sub);
