    Type Definitions

    JsonWebTokenPayload

    JSON Web Token payload that contains the claims.

    Type

    • Object

    Properties

    • aud String

      the token audience, e.g. acme.example.org.

    • exp Number

      the time at which the token expires, in seconds since Unix epoch.

    • nbf Number

      the time at which the token becomes valid, in seconds since Unix epoch.

    • sub String

      the authenticated subject, e.g. alice@example.org.

    See

    • https://tools.ietf.org/html/rfc7519#section-4.1

    Events

    DivaoApiLoaded

    DivaoApiLoaded event fires when the Divao API is ready and attached to the window object.

    Example

    window.addEventListener("DivaoApiLoaded", () => console.log("window.Divao can now be used"));

    DivaoSignOn

    DivaoSignOn event fires when a user has signed on or signed out.

    Example

    window.addEventListener("DivaoSignOn", () => console.log(Divao.JWT === null ? "user is not signed in" : "user is signed in"));
